Code

meme-simulator

A silly project with serious technologies I'm working on with another intern I met at Hyland in our spare time. It collects meme metadata ("memetadata") and attempts to generate memes using a Markov Chain model. To improve the model, I calculated a sentiment for certain words based on the upvote/downvote points of a meme, in the hopes that the Markov Chain could then pick more popular or funny words. We're looking into other NLP-based stuff so it should be fun to see where this goes! On the right (or below, if you're on mobile) is one of the best memes we have generated so far.

Node.js Proxy Server

I wrote a proxy server in Node.js while I worked at Hyland Software. I'm not privy to share too much code unfortunately, but I can talk about a few implementation details. Using Redis I was able to create a more efficient cache that their previous cache, and also was able to move it closer to the client with my Node.js server. This allowed for up to 60% speedups under heavy loads. These sorts of improvements are crucial for Hyland's goals of providing a cloud-based SaaS platform. This server was also able to sit on top of ASP.NET, WCF, and legacy .NET RPC/SOAP platforms while still providing significant performance increases. I was selected to present to representatives of each software development team at Hyland.

acupwith.me

My entry for Berkeley's CSUA Hackathon held from 11/21 - 11/22. I've noticed that a lot of people at cafes tend to be staring at screens rather than other people, so I got the idea to build this site with a friend to help bridge the gap. acupwith.me is a small web application built with Django and Postgres designed to help connect bored Berkeley cafe goers with others.

LearnCoin

I worked with a professor at Cleveland State University to create a resource to aid beginners with developing their own cryptocurrencies. LearnCoin is a fork of SmallChange, which itself is a fork of Litecoin. I found the experience of creating one's own cryptocurrency to be highly engaging and incredibly informative about how cryptocurrencies work, so I recommend those interested in cryptocurrencies to try building an altcoin for themselves!
